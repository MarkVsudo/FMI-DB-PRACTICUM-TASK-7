#### Running the project:

1. Install all the dependencies in client and server folders
2. Run client side `npm run dev`
3. Create a .env file inside server folder
   ```
   DB2_DATABASE=SAMPLE
   DB2_HOST=62.44.108.24
   DB2_PORT=50000
   DB2_USER=db2admin
   DB2_PASSWORD=db2admin
   DB2_SCHEMA=FN9MI0700339
   ```
4. Run server side `npm run dev`
